设计模式之策略模式

发表于 2016-08-29 | 分类于 技术笔记 ， Java | 阅读次数:

  策略模式，又叫算法簇模式。定义不同的算法簇，并且之间可以互相替换，此模式让算法的变化独立于使用算法的客户。也就是说把使用策略与对象独立出来，对象可以动态地改变对象的使用策略。

应用场景

迪仔，是一个吃货，喜欢吃雪糕还有 banana ，没错就是我。但是雪糕和香蕉有很多种，我想每一天换一种口味(哈哈哈)，并且把它记录在我的美食日记里。

接口

雪糕口味接口

public interface IIceCream {
    void writeIceCream();
}

香蕉类型接口

public interface IBanana {
    void writeBanana();
}

雪糕口味

香草口味实例

public class VanillaIceCream implements IIceCream {
    @Override
    public void writeIceCream() {
        System.out.println("香草味雪糕");
    }
}

草莓口味实例

public class StrawberryIceCream implements IIceCream {
    @Override
    public void writeIceCream() {
        System.out.println("草莓味雪糕");
    }
}

香蕉类型

皇帝蕉实例

public class EmperorBanana implements IBanana {
    @Override
    public void writeBanana() {
        System.out.println("皇帝蕉");
    }
}

苹果蕉实例

public class AppleBanana implements IBanana {
    @Override
    public void writeBanana() {
        System.out.println("苹果蕉");
    }
}

日期类

抽象日期类

public abstract class UnKnownDay {

    protected String day;
    protected IIceCream iIceCream;
    protected IBanana iBanana;

    protected UnKnownDay setIceCream(IIceCream iIceCream){
        this.iIceCream = iIceCream;
        return this;
    }

    protected UnKnownDay setIBanana(IBanana iBanana){
        this.iBanana = iBanana;
        return this;
    }

    protected void writeIceCream(){
        iIceCream.writeIceCream();
    }

    protected void writeIBanana(){
        iBanana.writeBanana();
    }
}

迪仔日期类，当然这里还可以自定你的日期类，然后就可以记录你的美食日记了

public class DiZaiDay extends UnKnownDay{

    public DiZaiDay(String day){
        this.day = day;
        System.out.println(day+",我品尝了");
    }
}

美食日记

public class FoodDiary {
    public static void main(String[] args) {

        DiZaiDay diZaiDay;
        diZaiDay = new DiZaiDay("8月29日");
        diZaiDay.setIceCream(new VanillaIceCream())
                .setIBanana(new EmperorBanana());
        diZaiDay.writeIBanana();
        diZaiDay.writeIceCream();

        diZaiDay = new DiZaiDay("8月30日");
        diZaiDay.setIBanana(new AppleBanana())
                .setIceCream(new StrawberryIceCream());
        diZaiDay.writeIBanana();
        diZaiDay.writeIceCream();
    }
}

  当然，你也可以设置你的的美食，比如番茄、曲奇饼干、咖啡等等，写完接口和美食实例并在日期抽象类中添加美食的接口调用就可以了。是不是很简单，你懂了么？
