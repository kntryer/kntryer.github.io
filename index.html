2016 年度总结

发表于 2016-12-28 | 分类于 随笔 ， 年度总结 | 阅读次数:

  去年11月份，在工作室的指导老师的介绍下，我们工作室几个人承接了一个APP的外包。没有任何外包经验的我们摸爬滚打，常常开几个钟的会来讨论功能需求，讨论进度。

  学习新的东西，时间总是很短。一月份很快就来临，因为经验的问题，我们的开发周期延长到这个月。半吊子 Android 的我在学东西很慢，很多东西都不理解，遇到 bug 的时候，一天很快就过去了。没有做过的东西都会觉得很难，但坚持下去一定会有收获。

  中旬的时候，忍受不了老妈的催促，我回家了。因为那时候奶奶的身体有恙，我想回去看看。在家的每天都会抽空看看奶奶还有爷爷，他们蛮开心的，我想老年时最盼的是儿孙的关心与陪伴吧。

  不得不说，我还挺喜欢瞎搞的。那时家里没有拉网线，搞开发变得很艰难。刚好家里有个路由器，就把邻居的 WiFi 覆盖到咱家来，虽然网速不稳定，但至少能查查资料，看看博客。我对编程这么感兴趣，可能很大部分来自于瞎搞，嘿嘿。

  二月份在忙碌的气氛中悄悄到来，整个埠头都忙着搞卫生、买年货、贴对联等等，脸上布满幸福的笑容。忙碌是春节的气氛吧，突然这样想。每年除夕的那晚我都没睡，静静的看着空白的天花板，放空自己，让思绪自动流转。看起来毫无意义，但很舒适。

  有相聚必有离别。在离开的那天，看着奶奶浑浊的双眼，有点不舍，我握住奶奶的手，对她说：“我过完这学期就回来看你，很快”。来到了九月，没想到这一别就是永远。

  三四月在忙一个大学生软件设计比赛，和舍友一起进行开发，那时我开始学习php接口开发，结果看了一周的php就开始打代码了。一开始毫无头绪的，不理解什么是接口。最后在慕课网上看了些视频，才知道接口和数据传输的相关知识。当然，也参考了邹锋师兄写过的代码，但是很多Sql语句看不上很懂（没怎么认真听过课的人啊，说真的，很吃亏）。除了这个比赛，还有很多有趣的事情，比如环创工作室开创学习部还有不定时授课，当时我还说了句很装逼的话，“没有行动，世界知识个观念”。

  五月六月就是毕业季和期末考试的组合，逃课到广州参加老哥毕业照，尴尬的是碰到班主任，差点被抓到了。当然还有环创大佬们的毕业照，烧烤等等。考完试之后，之前和老弟约好要去张家界玩，至于张家界游玩再写一篇博文吧，觉得有点亏，不过第一次组织旅游还挺开心的吧。

  旅游完，就要去找实习了。在广州疯狂地投简历和面试，还记得当时，面试了几家要不是没消息就是工资太低，心情很不好很烦躁。面试也变得很糟，没有来过通知。在家安静地想了一个星期，我觉得我不应该给自己太大压力，有面试就去面，没有面试就在家里学习。最后，八月多的时候，我去上班了，有家小公司招了我，很开心，这是我第一份工作呀。老板人很好，脾气也不错。

UDP协议详解

发表于 2016-09-18 | 分类于 技术笔记 ， Java | 阅读次数:

UDP 简称 User Datagram Protocol (用户数据报协议)，是一种无连接、不可靠的协议，每个数据报都是一个独立的信息，每个被传输的数据报很小（限定在 64k 以下），位于传输层 ( ip 位于网络层 ) 。而且 UDP 协议是不分客户端和服务端，只分发送端和接收端。因为 UDP 协议是面向无连接的，所以速度很快，但会出现数据包丢失。一般用于视频聊天、网络游戏等实时性较高的情况。
